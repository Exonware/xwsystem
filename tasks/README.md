## xwsystem tasks

This folder contains **public task descriptions** for xwsystem. Each task is a self‑contained exercise that a developer or AI agent can attempt, using only the repo and its docs as inputs.

You can use these tasks to **practice working with xwsystem**, to **evaluate a freelance engineer**, or to **benchmark an AI agent** on realistic maintenance and feature work. Tasks should be concrete, falsifiable, and small enough to complete in a few hours.

When you add a new task, treat it like a mini spec: explain the goal, constraints, and acceptance criteria, and link to any relevant REFs, guides, or modules in this project.
