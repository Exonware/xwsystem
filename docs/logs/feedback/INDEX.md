## Routing rules (summary)

- **New feature request** → `xwsystem/docs/REF_IDEA.md` then requirements in `xwsystem/docs/REF_PROJECT.md`
- **Bug** → reproduce (`GUIDE_52_DEBUG.md`), fix (`GUIDE_53_FIX.md`), record change under `xwsystem/docs/changes/CHANGE_*.md`
- **DX friction** → update `xwsystem/docs/REF_DX.md` and route to IDEA/DEV/FIX as appropriate
- **Performance regression** → benchmarks (`REF_BENCH.md` + `GUIDE_54_BENCH.md`)
